Для работы программы Persona_1.0 необходимо:
	1. Установить интерпретатор Python версии 3.3.5 и выше
	2. Установить PyQt4 и выше

Вместо двух указанных выше пунктов можно установить WinPython, скачав и запустив файл:
	WinPython-64bit-3.3.5.0.exe
или
	WinPython-32bit-3.3.5.0.exe
или выбрав версии выше

3. Вынести на рабочий стол ярлыки файлов:
	Persona_1.0.pyw
	Persona.doc
	Persona.pdf (предварительно создать)

4. Запустить файл create_database.py

5. Запустить файл insert_admin.py

6. Наслаждаться использованием, запуская файл Persona_1.0.pyw
